USB(uart)Command: 

1. Enable Raw data output
```
d;  
```
2. Disable Raw data output
```
D;  
```
3. Switch to PA mode:
```
l;   
```
4. Switch to Probe(endstop) mode:
```
e;
```
5. set thrhold:xx; xx must be number, recommend number is 3 to 10. for example, we want to set the value to 8
```
8;  
```
Note: if you use other Com Port software, remember to add the end character "\n"
